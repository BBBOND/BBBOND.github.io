2016总结

0x00 工作篇

2016年，结束了大三的在校学习生活，开始了大四外出实习修行之路。经过一番准备以及多次面试之后，我来到了现在的实习公司，到2017年1月末度过了6个月漫长的实习。2017年准备去别的公司闯一闯。

0x01 姿势篇

对于2016年新学到的知识技能方面，主要来源于上半年的自学以及下半年公司内的实践学习，大概总结了一下：

  1. angularJS: 在学校时做一个小项目(基于Ionic)用到这个
  2. SSH: java web 三大框架，16年初自学过，然后上课也讲过点，但是后来由于一直没用，现在手已经很生了
  3. 看过一点Lucene搜索引擎(写了个小Demo)
  4. NoHttp: 读了部分源码，然后基于这个库的1.0.4版本进行了一次封装，主要包含String、Json、JsonArray、Image的请求以及下载请求。
  5. 抄了一份Android多线程下载的代码到自己的github
  6. 搭建了自己的博客，虽然还没写几篇博客
  7. ReactNative / React.js / Redux: 在公司学到的新东西
  8. GitFlow: 在公司学到的，可以有效的实现多人协同开发
  9. charles: 一个很有用的抓包工具，网络调试相当给力
  10. MVP: Android开发的一个代码设计模式，可以有效地理清代码逻辑，并且也方便了单元测试

0x02 项目篇

  1. 长春大学教务处: 突发奇想写了个学校教务处的移动端
  2. 《科学脱口秀》: 喜欢听他们的节目，但是又不想下载广告推送多如毛的第三方app，因此索性自己写一个(哈哈，当个程序员的好处)，之后看完成度推荐给节目试试。(正在开发)
  3. 长春大学教务处2.0: 由于教务处改版，加上对第一版app样式，以及对代码架构的不满意，因此重新写了一遍(正在开发)

0x03 生活篇

在自己租房之后，由于下班晚、晚上回家还要做自己的项目、偶尔学习学习新知识，回想起来竟然每天2点多睡已经快5个月了。真该好好休息休息，注意身体了。

另外，在实习期间我的体重猛增，，已经胖得令人发指了，减肥！
